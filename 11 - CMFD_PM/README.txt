Date released 6/07/2015, version 1.0.
Matlab functions for the detection and localization of copy-move forgeries with the technique described in 
"Efficient dense-field copy-move forgery detection", 
written by D. Cozzolino, G. Poggi and L. Verdoliva, 
IEEE Trans. on Information Forensics and Security, in press, 2015.
Please refer to this paper for a more detailed description of the algorithm.
